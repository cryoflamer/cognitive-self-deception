# Cognitive Self-Deception

A small research project on how intelligence can reinforce self-deception through motivated reasoning, cognitive dissonance, rationalization, intellectualization, and selective reflection.

## Core thesis

Intelligence does not eliminate bias — it can optimize it.

The project explores a paradox: higher cognitive complexity can improve epistemic accuracy in some contexts, but it can also make self-deception more sophisticated, stable, and rhetorically convincing when reasoning is motivated by identity, self-esteem, or psychological comfort.

## Research focus

The project is centered on several related mechanisms:

- motivated reasoning
- cognitive dissonance
- rationalization
- intellectualization
- selective reflection
- metacognitive blind spots
- inversion of critical theories into defensive tools
